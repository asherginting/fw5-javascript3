## Installation

1. Clone This Repo

```sh
git clone https://github.com/asherginting/fw5-javascript3
```

2. Go To Folder Repo

```sh
cd fw5-javascript3
```

3. Open Project, Run Node .task..js in terminal.
4. For Task 4, Npm install to add package and ```npm run start```
